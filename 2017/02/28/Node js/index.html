Node.JS文件操作

  • Node.JS

Node.JS的一些基本的文件操作

一、文件操作

1.1 相关模块

1： Path 路径字符串操作
2： Fs 文件操作
3： Readline 按行读取文本文件
4： 第三方模块 fs-extra

1.2 path模块

1： path.basename(path[, ext]) 注意第二个参数的用户，获取文件名
2： path.dirname(path) 获取文件目录
3： path.extname(path) 获取文件的扩展名
4： path.format(pathObject) 和parse(pathString)
5： path.isAbsolute(path) 判断是否是绝对路径
6： path.join([…paths]) 拼接路径字符串
7： path.normalize(path) 将非标准路径转换为标准路径
8： path.sep 获取操作系统的文件路径分隔符

1.3 fs模块

Fs模块对文件的几乎所有的操作都有同步和异步两种形式。
参考：http://wenku.baidu.com/view/fbcf1bb7c77da26925c5b015.html?from=search

1.3.1 同步异步读取文件

1： fs.readFile(file, data, [,option], callback)
2： fs.readFileSync(file, data, callback)

1.3.2 同步异步写文件

1：fs.writeFile(file, data, [,option], callback)
2：fs.writeFileSync(file, data, callback)

1.3.3 同步异步向文件中追加内容

1：fs.appendFile(file, data, [,options], callback)
2：fs.appendFileSync(file, data, [,options])

1.3.4 其他文件操作

1：Fs.exists(path, callback(exists)) //验证是否存在
2：Fs.stat(path, callback(err, stats))//获取文件信息
3：Fs.unlink(path, callback) //删除文件
4：Fs.rename(oldPath, newPath, callback) 重命名文件或者目录
5：Fs.rename(oldPath) 移动文件

1.3.5 目录操作

1：Fs.mkdir(path, [,model],callback) //创建一个目录
2：Fs.rmdir(path, callback) //删除一个目录
3：Fs.readdir(path, callback) //读取一个目录

1.3.6 监视文件

 fs.watchFile(filename, [,options], listener(curr, prev))
