Hexo + GitHub搭建个人博客

  • 宅男技术

本教程只是简单的介绍,估计从来没有用过的会看不懂,没关系，欢迎来请教。
稍微懂点的,应该会一看就懂的ヾ(ﾟ∀ﾟゞ)ヾ

Quick Start

基础准备

1: 安装Node.js

  • 到Node.js官网下载,并按照步骤安装 https://nodejs.org/en/
  • node -v
  • 出现版本号，说明安装成功

2:安装Git

  • 下载安装Git https://git-scm.com/downloads
  • git –version
  • 出现版本号，说明安装成功
  • 创建reposity
    在GitHub上创建reposity（没有Github的去注册一个）,注意,Repository name 是一个博客的名字,比如我的:my-blog

3:安装Hexo

  • 桌面打开命令窗口执行 npm install -g hexo
  • 安装成功后在任意一个盘新建文件夹(最好英文),命令窗口执行 hexo init
  • 本地服务启动 hexo server
    浏览器输入 http://localhost:4000

同步博客到github

1：修改文件

  • 博客根目录下找到_config.yml文件
  • 用Sublime Text2之类的程序打开修改
  • deploy:
    type: git
    repository:**(你Github上创建的reposity地址)
    branch: gh-pages

2：然后执行如下命令

  • npm install hexo-deployer-git –save

3：再执行

  • hexo deploy

常用命令

1: generate 生成静态文件

  • hexo g

2: server 启动服务器

  • （成功后可以查看本地的更改后的效果http://localhost:4000/）
  • hexo s

3: deploy 部署网站。部署网站前，需要预先生成静态文件（hexo g）

  • hexo d

4: clean 清除缓存文件（db.josn）和已生成的静态文件（publick）。(如果部署后没有发生改变，可以下面命令清除缓存后，就可以看到效果了)

  • hexo clean
